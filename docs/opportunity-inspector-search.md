# Opportunity inspector and search slice

This issue #54 slice adds client-side opportunity search and sorting, and moves detailed evidence/workflow content from inline row expansion into a focused inspector drawer.

## Scope

- search title, company, and location;
- sort by score, title, or company;
- reset pagination after query changes;
- preserve the compact queue and decision selector;
- open one opportunity in an accessible dialog-style inspector;
- preserve readiness, source, preparation, and application actions.

## Boundaries

No backend contract, evaluation rule, capture runtime, or application model changes are included.
